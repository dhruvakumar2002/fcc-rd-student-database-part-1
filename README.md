Student Database - Part 1 

This project is part of the freeCodeCamp Relational Database Certification. It focuses on building a PostgreSQL database from scratch, using Bash scripts to automate data insertion, and managing a relational schema for student and course information. 

The goal of this project was to learn how to:

Create a relational database structure using PostgreSQL.
Write Bash scripts to read data from CSV files and insert it into a database.
Manage primary and foreign key relationships between tables (students, majors, courses, and majors_courses).
